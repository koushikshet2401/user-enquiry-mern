# User Enquiry MERN App

A full MERN stack CRUD application with an enquiry form and admin table.

## Features
- Create enquiry
- View enquiry list
- Edit enquiry
- Update enquiry
- Delete enquiry
- SweetAlert confirmation
- Toast notifications

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- Tailwind CSS
